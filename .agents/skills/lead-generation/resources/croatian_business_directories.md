# Croatian Business Directories

## Online Directories for Supplementary Scraping

These directories can be scraped using Apify's generic web scraper to find businesses NOT listed on Google Maps.

### Primary Directories

| Directory | URL | Notes |
|-----------|-----|-------|
| **Zlatne Stranice** (Yellow Pages) | [zlatnestranice.hr](https://www.zlatnestranice.hr) | Croatia's official Yellow Pages. Search by category + city. Best coverage. |
| **Njuškalo** | [njuskalo.hr](https://www.njuskalo.hr) | Croatia's largest classifieds. Has a business directory section. |
| **Poslovni Imenik** | [poslovni-imenik.com](https://www.poslovni-imenik.com) | Business directory with company details. |
| **Bisnode** | [bisnode.hr](https://www.bisnode.hr) | Company database with financial data (paid for detailed info). |

### Category Names (Croatian)

When searching Croatian directories, use these category names:

| English | Croatian Category |
|---------|------------------|
| Barber shop | Brijačnice |
| Hair salon | Frizerski saloni |
| Beauty salon | Kozmetički saloni |
| Hairdresser | Frizeri |
| Nail salon | Saloni za nokte |

### Social Media as Directories

Many Croatian businesses ONLY have social media presence:

| Platform | Search Method |
|----------|--------------|
| **Facebook** | Search `frizerski salon Rijeka` in Facebook search bar |
| **Instagram** | Search hashtags: `#frizerrijeka`, `#barberrijeka` |
| **Google Business** | Already covered by Google Maps scraper |
